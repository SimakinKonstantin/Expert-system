# Экспертная система - помощник по выбору монитора.
Проект на Prolog SWI, который подбирает компьютерный монитор в зависимости от сценария использования.

## Функционал
* Поэтапный опрос о необходимых характеристиках монитора;
* Логический вывод результата на основе фактов и правил;
* Вывод цепочки, которая привела к результату;
* Между консультациями запоминается выбранные варианты, получают более высокий приоритет при следующих итерациях.

## Инструкция по запуску
1. Через cli: `swipl -s .\consult.pl -g start -t halt`;
2. Через онлайн IDE (https://swish.swi-prolog.org/): в поле ввода ввести `start` и запустить.
